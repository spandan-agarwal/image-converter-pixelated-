hello there,
this is a minor fun project for learning the basic use and understanding of pyautogui and turtle module.
it converts a normal image into pixelated image.
(NOT USER FRIENDLY)
(TIME CONSUMING)

REQUIREMENTS-->
python(3.8 or later recommended),
python turtle module (pip3 install turtle)
python pyautogui module (pip3 install pyautogui)


USAGE-->
step-1>run the program in python terminal
step-2>point the cursor on the top left corner of the image and wait until it marks the coordinates.
step-3>point the cursor on the bottom right corner of the image and wait until it marks the coordinates.
step-4>wait for about 20 minutes for the image to complete.
